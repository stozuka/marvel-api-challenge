# Marvel API Challenge

## Setup

1. Install dependencies

```bash
npm i
```

2. Setup husky pre-commit hook

```bash
npm run prepare
```

## Start the server in local

- Watch mode

```bash
npm run start:dev
```

- Normal mode

```bash
npm run start
```

## Run the tests

- Unit test

```bash
npm run test
```

- E2E (integration) test

```bash
npm run test:e2e
```
